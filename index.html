About Me

Hello there! I'm Chris. I'm a strategic introvert who enjoys wrestling with code to build cool things. It all started with the Joe Rogan Podcast, then Brazilian Jiu Jitsu, then the Lex Fridman Podcast, and then this thing called Javascript! The journey has been endless from there. My need for challenge and skill climbing has become satiated 😄.

Everyday is a new challenge and I embrace that!
