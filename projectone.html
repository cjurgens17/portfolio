Pool Vikings

Pool Vikings is a static business website built with Bootstrap, Java, Spring, and Thymeleaf. Being in the Pool Industry myself for a long time, I really wanted to design something that I felt would be a nice catch to the normal pool customer. This project required a lot of bootstrap research to redo the template in ways that I felt it should look.

Project Goal and Purpose

This Project was intended to attract and route users to a contact line by providing them with information about the companies services and culture. For this reason the architecture of the project revolved mainly around different pool departments. From there the atmosphere of the website was designed to invite a cooling and joyful environment, so customers would have no hesitation in reaching out to the office contact line.

Web Stack Choices

This project was mainly abouting learning bootstrap, and creating the ability to understand and use templates. Java and Spring were chosen for quick deployment, which easiy allowed routing between views through an MVC design pattern.

Thymeleaf was implemented because its an excellent option for synergy with Java and Spring with its server-side template engine. This allowed for good performance and a quick transition between views.
